# 🚀 TaskFlow — Modern Task Management Web App

TaskFlow is a modern, interactive task management web application built using HTML, CSS, and JavaScript.
It demonstrates real-world UI/UX concepts and Human-Computer Interaction (HCI) principles through a clean, responsive interface.

---

## ✨ Features

* 🧩 Drag & Drop Kanban board
* ✏️ Inline task editing (direct selection)
* 🖱️ Right-click contextual menu (Edit, Delete, Complete)
* 📦 Modal overlay for adding tasks
* 🔄 Virtual page navigation (SPA-like tabs)
* 🌙 Dark mode & personalization settings
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom UI Design, Flexbox, Grid)
* JavaScript (Vanilla JS, DOM Manipulation)

---

## 🎯 HCI Concepts Implemented

* Visibility
* Feedback
* Consistency
* Affordance
* Direct Manipulation
* Contextual Interaction
